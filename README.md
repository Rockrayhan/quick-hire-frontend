##  QuickHire Task – Frontend

A simple client site for Job website — built with **React**, **TypeScript**, and **Shadcn UI**.

### Frontend Live link: https://quickhire-frontend-lake.vercel.app/


#### ✨ Features :

- Responsive Home page based on Figma design
- Job listing / All Jobs page with search and filtering
- Job details page with apply form to apply on the job
- Basic Admin view page with Add new job & Delete job


#### ⚙️ How to Set Up Locally

- Make sure **Node.js** and **Git** are installed on your machine.

- then git clone the repository `git clone https://github.com/Rockrayhan/quick-hire-frontend`

- Navigate to the project folder and run: - `npm i` then `npm run dev`

- Copy `.env.example` to `.env` and add the following:
```VITE_BASE_URL=https://quick-hire-api.aangonstudio.com/api```

- to go the url `http://localhost:5173/` in your browser.
